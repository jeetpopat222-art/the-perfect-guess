# 🎯 Number Guessing Game

> *Can you crack the code before your attempts run out?*

---

## 🕹️ What is this?

A terminal-based number guessing game built in Python. The program secretly picks a random number between **1 and 100** — your job is to guess it in **10 attempts or less**. Every wrong guess gives you a hint. Run out of attempts? Game over. Nail it? Glory is yours.

---

## ⚡ Features

- 🎲 Random number generated every round (1–100)
- 🔼 / 🔽 Hints after every wrong guess (`Higher` / `Lower`)
- 🔢 Attempt counter so you always know how many chances are left
- 💀 Game over screen reveals the secret number
- 🔁 Play again without restarting the program

---

## 🚀 How to Run

Make sure you have Python installed, then:

```bash
python guessing_game.py
```

No external libraries needed. Just pure Python. 🐍

---

## 🎮 How to Play

```
enter a number to guess: 50
Lower number please
9 attempts remaining

enter a number to guess: 25
higher number please
8 attempts remaining

enter a number to guess: 37
you win! 🎉
```
